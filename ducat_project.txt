Key Features Implemented:
Multi-batch Selection: Select multiple batches from dropdown
Message Templates: Pre-defined templates for common scenarios
Preview Functionality: See message and student count before sending
Bulk Sending: One-click send to all students in selected batches
Confirmation Dialog: Prevent accidental sends

How to Use:
Select one or more batches from the multi-select dropdown
Choose a template or write custom message
Click "Prepare Message" to see preview
Click "Send Messages" to deliver to all students
